# Installing

Install global NPM pacages

```
npm install -g jspm
```

Install local NPM Packages

```
npm install
```

Install JSPM packages using the base config

```
cp .jspm.config.js jspm.config.js
jspm install
```

# Running

Run each line in a new terminal tab

```
npm run bundle
npm run serve
```

Open the site at `127.0.0.1:3000`
